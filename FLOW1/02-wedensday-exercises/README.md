## Onsdag

Her ligger alle "Exercises" samlet i en enkelt opgave, der ligger i denne mappe kaldet rest-erxercises.

Alle dele blev dækket og der kunne tilgåes forskellige sider på localhost som fremviste de forskellige ting. 
F.eks "http://localhost:8080/rest2/api/animals_db/animals" ville fremvise alle dyrene i min DB. 

I denne opgave blev vi introduceret til REST, hvor vi lærte om GET annotationen.
Dette gjorde at vi via vores DB og localhost kunne navigere rundt og derved kalde forskellige GET request som hentede forskellige ting fra DB'EN.
